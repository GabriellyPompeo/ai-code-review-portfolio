## 🔍 Review Methodology

Each code review in this portfolio evaluates AI-generated code across these dimensions:

1. **Correctness** — Does the code solve the stated problem?
2. **Efficiency** — Time and space complexity analysis
3. **Best Practices** — Code style, naming conventions, readability
4. **Edge Cases** — Handling of unexpected inputs and boundary conditions
5. **Security** — Potential vulnerabilities or unsafe practices
6. **Instruction Following** — Did the AI follow the prompt requirements?

```markdown
## Review: [Title]

### ✅ What the AI did well
- Clear variable naming
- Correct base case handling

### ❌ Issues Found
- Bug: Off-by-one error on line 14
- Missing input validation for negative numbers
- O(n²) complexity where O(n log n) is achievable

### 💡 Suggested Fix
[improved code snippet]

### 📝 Final Feedback
[structured written feedback]
```
